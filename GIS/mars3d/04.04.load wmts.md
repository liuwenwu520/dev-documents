# 加载 wmts

## 加载 geoserver wmts 服务

- 要求

```
1.wmts 要求切片策略的参考坐标系为 EPSG:4326（WGS84）

2.cesium 只支持 rest 风格的 wmts 链接，可在 GetCapabilities 中查看 rest 链接
```

### 在初始化 map 后添加 wmts 图层

- 示例

```js
const maxLevel = 21;
const tileMatrixPrefix = 'EPSG:4326:';
const tileMatrixList = new Array(maxLevel)
  .fill(0)
  .map((item, i) => (item = tileMatrixPrefix + i));
const wmtsLayer = new mars3d.layer.WmtsLayer({
  url: 'http://192.168.7.18:8710/geoserver/aaa/gwc/service/wmts/rest/bc_yjjbnt/{style}/{TileMatrixSet}/{TileMatrix}/{TileRow}/{TileCol}?format=image/png',
  style: 'aaa:AA_BC_YJJBNT',
  tileMatrixSetID: 'EPSG:4326',
  tileMatrixLabels: tileMatrixList,
  crs: 'EPSG:4326',
  flyTo: true,
});
map.addLayer(wmtsLayer);
```
