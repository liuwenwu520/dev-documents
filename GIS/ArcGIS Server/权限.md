## 权限
### # 获取令牌
> [获取 ArcGIS Server 令牌](https://enterprise.arcgis.com/zh-cn/server/latest/administer/windows/acquiring-arcgis-tokens.htm)
- url
```
http://localhost/arcgis/admin/generateToken
```
- method
```
post
```
- params
```js
const data = {
  // 用户名
  username: 'admin',
  // 密码
  password: 'admin',  
  // 返回格式
  f: 'json',
  // 客户端类型 参考
  client: 'referer',
  // 参考配置 webapp 地址
  referer: 'http://localhost',
  // 有效时间 (minutes)
  expiration: 60
}
```
