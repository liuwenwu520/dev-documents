# 网络管理
## 网络工具箱
### # iproute2
> 新版网络管理工具箱，目前在维护  
> 包含 ip 命令
```
sudo apt install iproute2
```
### # net-tools
> 旧版，2001年停止维护  
> 包含 netstat、ipconfig 命令
```
sudo apt install net-tools
```

ip [OPTIONS] OBJECT { COMMAND | HELP }
ip [-force] -batch <filename>
