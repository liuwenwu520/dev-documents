## docker statck
管理 Docker 栈
- syntax
```
docker stack [options] <COMMAND>
```
- options
```

docker statck ls
```
### # docker stack deploy
> 发布一个新栈或者更新一个已存在的栈
### # docker stack ls
> 列出所有栈
### # docker stack ps
> 列出栈中运行的任务
- syntax
```
docker stack ps [<opitons>] <stack>
```
### # docker stack rm
> 移除一个或多个栈
### # docker stack services
> 列出栈中的服务
- syntax
```
docker stack services [<opitons>] <stack>
```
