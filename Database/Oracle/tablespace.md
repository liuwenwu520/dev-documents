## 表空间

### 表空间数据占用情况
```sql
SELECT TABLESPACE_NAME, SUM(BYTES)/1024/1024 AS "SIZE(M)" FROM SYS.DBA_SEGMENTS WHERE TABLESPACE_NAME = 'SDE_TBS' GROUP BY TABLESPACE_NAME;
```
### 修改表空间自动扩展无上限
```sql
alter tablespace sde_tbs autoextend on next 100m maxsize unlimited; -- 会报错：不支持对小文件表空间 SDE_TBS 的操作
ALTER DATABASE DATAFILE 'D:\APP\ADMINISTRATOR\PRODUCT\11.2.0\DBHOME_1\DATABASE\SDE_TBS' AUTOEXTEND ON NEXT 200M MAXSIZE UNLIMITED;
```
