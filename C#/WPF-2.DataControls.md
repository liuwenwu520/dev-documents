## 数据控件
### 1 DataGrid
> 表格数据空间

- Start
```xml
<DataGrid>
  Items
</DataGrid>
```

- AutoGenerateColumns
```
prop 自动生成列
```

- ItemsSource
```
prop 绑定资源，如：DataTable
```

- IsReadOnly
```
prop 只读，默认为false
```

- SelectionMode
```
prop 选择模式，默认扩展
单行 Single
扩展，可选择多行 Extended
```

- GridLinesVisibility
```
prop 网格线，默认全部显示
```
